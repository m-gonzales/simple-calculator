# simple-calculator

Calculator App — Java Swing

A modern, fully‑functional calculator built using Java Swing, featuring a clean UI, expression evaluation, functional parentheses, and a custom‑designed history window.

- Standard arithmetic operations
- Functional parentheses ( )
- Square root √
- Percentage %
- Sign toggle +/-
- Delete (backspace)
- calculation history
